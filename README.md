# 🛍️ Zenvy Store

Zenvy Store is a modern and responsive online store homepage built using HTML, CSS, and JavaScript. It showcases products in different categories with interactive filtering and a clean UI design.

---

## 🚀 Project Overview

This project is developed as part of an internship task to create a dynamic online store homepage. The focus is on UI/UX design, interactivity, and responsive layout.

---

## ✨ Features

- 🧾 Product cards with image, name, and price
- 🗂️ Category-based filtering (Electronics, Fashion, Books)
- 🛒 Add to Cart button (basic interaction)
- 🎨 Modern UI with gradient background
- 📱 Responsive grid layout
- ⚡ Dynamic data loading using JSON (mock API)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM Manipulation & Fetch API)
